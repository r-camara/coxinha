# Spec 0029: Unified daily timeline

- **Status:** draft
- **Phase:** F3
- **Owner:** Rodolfo
- **Depends on:** 0005, 0026, 0027, 0028
- **Relevant ADRs:** —

## Why
"Open the app, see what I have to do today" — without opening five
windows.

## Scope

### In
- "Today" view aggregating:
  - Daily note
  - Scheduled meetings (spec 0024/0025)
  - Open tasks for the day (spec 0026)
  - Current PRs/issues (spec 0027, 0028)
- Ordered by time when applicable, otherwise by priority

### Out
- Time blocking / pomodoro → F5
