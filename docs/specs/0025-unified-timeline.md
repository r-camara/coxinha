# Spec 0025: Unified daily timeline

- **Status:** draft
- **Phase:** F3
- **Owner:** Rodolfo
- **Depends on:** 0002, 0022, 0023, 0024
- **Relevant ADRs:** —

## Why
"Open the app, see what I have to do today" — without opening five
windows.

## Scope

### In
- "Today" view aggregating:
  - Daily note
  - Scheduled meetings (spec 0020/0021)
  - Open tasks for the day (spec 0022)
  - Current PRs/issues (spec 0023, 0024)
- Ordered by time when applicable, otherwise by priority

### Out
- Time blocking / pomodoro → F5
