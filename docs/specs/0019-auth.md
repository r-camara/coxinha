# Spec 0019: Auth (Keycloak / JWT)

- **Status:** draft
- **Phase:** F2
- **Owner:** Rodolfo
- **Depends on:** spec 0015
- **Relevant ADRs:** —

## Why
Sync implies multi-device, which implies identity.

## Scope

### In
- Option 1: self-hosted Keycloak (full OIDC)
- Option 2: plain JWT with local registration (email + password)
- Client-side token refresh
- Device trust list (revoke a lost device)

### Out
- Social SSO (Google, GitHub) → F3
